Ways to use the apps

These are launch methods for the same GitHub Pages applications, not separate builds with different functionality.

1

Regular web page

Nothing to install. Browser controls remain visible. Best for opening a shared link or testing changes immediately.

Choose an application above and open it normally.

2

Installed PWA

Creates a launcher icon and opens in a standalone window. The application remains the same live site and can work offline after its first successful load.

Open the app in Chrome, then choose Install app or Add to Home screen.

3

Lightweight wrapper

Use an app URL in Native Alpha or another wrapper when you want no normal browser interface without maintaining a separate APK.

URL pattern:
evs625.github.io/mobile-apps/apps/<app-name>/

Main distinction: browser page, installed PWA, and wrapper are different windows for the same application—not separate application versions.
